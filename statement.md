# Problem Statement 

Many people struggle to maintain mental-wellbeing along with physical -wellbeing.  
People usually don’t go to a therapist as it is still considered a social taboo .  
People need an easy to use ,simple platform that uplifts their mood ,provides help and counselling . 

# Scope of the program   
1)This program can be used by various therapist to provide an immediate help to the user that can act as a temporary solution .   
2)It can help in improving an individual’s mental health and make meaningful contributions to the society   
3)The report provided at the end can be useful in measuring the progress in the condition of the user  by the therapists .   

# Target Users 
 1)people of all age bracket that seek counselling 
2)hospital therapist to provide an immediate help to the individual .   
# High level features 
1)visualization : Graphical representation of comparision using matplotlib   
2)recommendation : basic algorithm that recommends sources that can be helpful.
