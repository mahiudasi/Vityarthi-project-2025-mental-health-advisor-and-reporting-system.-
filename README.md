* Project Overview

  
Mental health is not just an individual ,but a community concern The project aims to find support for anxiety, depression , stress , trauma , burnout etc . mental health is also important as it can affect a person’s physical health .


the objective of this project in to improve an individual’s mental health and make meaningful contributions to the society. 


* Features of the Project
 Provides words of affirmation to the user based on their state of mind 
 suggests useful application , articles and books to the user .
 allows user to directly open the suggested application ,book or article 
 Provides a my feeling report at the end as a tracker.

  
* Tools Used 
 Language: python 3.13 
 visualization :matplotlib(for pie plot ) 
 webbrowser  module to open a specific url

  
* Steps to run the project 
   step 1 - the user is asked how are you feeling today . enter your current  feelings/mood.   
   step 2 - the user is provides useful app/article/book . enter if you want to open the url now with yes or no  
   step 3 - the user is asked if they want to get their “my feeling report” ,later provides a pie chart that showcases a monthly analysis of the individuals feelings/mood.
